Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)!

Nessa frase temos um termo em destaque usando a tag <strong> (semântica)!

Itálico / Ênfase

Nessa frase temos um termo em Itálico usando a tag <i> (não semântica)!

Nessa frase temos um termo em Ênfase usando a tag <em> (semântica)!

Texto marcado

Nessa frase temos um termo marcado usando a tag <mark>!

E no outro parágrafo, temos outro termo marcado no final!

Texto grande e texto pequeno

Estamos criando um texto grande com a tag <big> e um texto pequeno com a tag <small>!

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado com a tag <del>!

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins> (semântica)!

Existe também o texto sublinhado com a tag <u> (não semântica)!

Texto sobrescrito

Para inserir coisas do tipo x20+3 com a tag <sup>!

Texto subscrito

Para inserir coisas do tipo H2O com a tag <sub>!
